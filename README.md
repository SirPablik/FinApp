#  FinPlan.Web - Управление личными финансами

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
[![ASP.NET Core](https://img.shields.io/badge/ASP.NET%20Core-8.0-512BD4?logo=dotnet)](https://dotnet.microsoft.com/)

**FinPlan.Web** — это современное веб-приложение для учёта личных финансов, планирования бюджетов и управления инвестициями.

## Возможности

### Учёт финансов
- **Доходы и расходы** — полный контроль над финансами
- **Категоризация** — группировка транзакций по категориям
- **Фильтры и поиск** — быстрый поиск по описанию, дате, категории
- **Экспорт в CSV** — выгрузка данных для отчётности

### Бюджетирование
- **Планирование бюджетов** — установка лимитов по категориям
- **Контроль расходов** — автоматический подсчёт потраченного
- **Уведомления** — предупреждения при 80% и 100% использования бюджета
- **Визуализация** — прогресс-бары и цветовая индикация

###  Инвестиции
- **Учёт активов** — акции, криптовалюты, облигации
- **Отслеживание стоимости** — общая стоимость портфеля
- **История покупок** — дата и цена приобретения

Аналитика и отчёты
-  **Дашборд** — сводная статистика
-  **Графики** — визуализация расходов по категориям и динамики
-  **Статистика** — баланс, доходы, расходы за месяц

### Интерфейс
- **Адаптивный дизайн** — работает на всех устройствах
- **Тёмная/Светлая тема** — переключение с сохранением
- **Sidebar навигация** — удобный доступ ко всем функциям
- **Bootstrap 5** — современный UI

### Главная страница
- **Курсы валют** — актуальные курсы от ЦБ РФ
- **Финансовые новости** — последние события из мира финансов

## Установка

### Требования
- [.NET 8.0 SDK](https://dotnet.microsoft.com/download/dotnet/8.0) или выше
- [Visual Studio 2022](https://visualstudio.microsoft.com/) (рекомендуется) или [VS Code](https://code.visualstudio.com/)
- SQL Server / SQLite

### Шаг 1: Клонируйте репозиторий

```bash
git clone https://github.com/SirPablik/FinApp.git
cd FinApp
```
### Шаг 2: Восстановите зависимости
```bash
dotnet restore
```
### Шаг 3:Настройте строку подключения
```json
"ConnectionStrings": {
  "DefaultConnection": "Server=(localdb)\\mssqllocaldb;Database=FinPlanDb;Trusted_Connection=True;MultipleActiveResultSets=true"
}
```
### Шаг 4: Создайте базу данных
```bash
# Создайте миграцию
dotnet ef migrations add InitialCreate

# Примените миграцию
dotnet ef database update
```
### Шаг 5: Запустите приложение
```bash
dotnet run
```
 Использование
1️ Регистрация
Откройте главную страницу
Нажмите "Регистрация"
Введите email и пароль
Нажмите "Зарегистрироваться"

2️ Добавление транзакции
Перейдите в раздел "Транзакции"
Нажмите " Добавить транзакцию"
Заполните форму:
Тип (Доход/Расход)
Категория
Сумма
Дата
Описание
Нажмите "Создать"

3️ Создание бюджета
Перейдите в раздел "Бюджеты"
Нажмите " Создать бюджет"
Укажите:
Категорию
Лимит суммы
Период (начало и конец)
Нажмите "Создать"

4️ Добавление инвестиции
Перейдите в раздел "Инвестиции"
Нажмите " Добавить актив"
Заполните:
Название актива (например, AAPL, BTC)
Тип (Акция, Криптовалюта, Облигация)
Количество
Цену покупки
Дату покупки
Нажмите "Создать"

5️ Просмотр статистики
Откройте "Дашборд"
Просмотрите:
Общий баланс
Доходы и расходы за месяц
Графики расходов по категориям
Динамику по месяцам
Последние транзакции

6️ Экспорт данных
Перейдите в нужный раздел (Транзакции/Бюджеты/Инвестиции)
Примените фильтры (опционально)
Нажмите " Экспорт в CSV"
Откройте файл в Excel

7️ Смена темы
Нажмите на кнопку  в боковом меню для переключения между светлой и тёмной темой.
